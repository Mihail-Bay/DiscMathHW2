### Пример графа:
```
    A
  / | \
 B  C  D
  \ | /
    E
```
### Вес рёбер:
- A-B: 1
- A-C: 2
- A-D: 3
- B-E: 4
- C-E: 5
- D-E: 6


### Пример ввода для данного графа:

```
{'A': {'B': 1, 'C': 2, 'D': 3}, 'B': {'A': 1, 'E': 4},
'C': {'A': 2, 'E': 5}, 'D': {'A': 3, 'E': 6}, 'E': {'B': 4, 'C': 5, 'D': 6}}
```

### Пример вывода:
```
Минимальное остовное дерево:
[('A', 'B', 1), ('A', 'C', 2), ('A', 'D', 3), ('B', 'E', 4)]
Общая стоимость: 10
```

Это означает, что минимальное остовное дерево состоит из рёбер A-B, A-C, A-D и B-E, а общая стоимость дерева равна 10.
